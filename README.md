This is a simple CLI Python based program I half made half vibe-coded to download manga from cubari.moe.

To use: Simply pip install -r requirements.txt (may be python3 -m pip install or python pip install, or just pip install depending on your setup) and then run the Python script.
This allows you to get either a PDF, or raw images.

I don't really have any plans to support others, but if this is useful to you please feel free to use it, since I couldn't find any similar tools that worked well for my use case.
